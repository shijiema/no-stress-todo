mobile app of to do task management. 

the home screen is vertically split into these sections. from top to bottom:
1.  arraw from left to right, with text time flys;
2. Today's Date
3.  a big rectangle split into 5 small boxes
3.1 top section is for what are in execution, call in "in execution"
3.1.1 each task is represented as a rectangle, could be bigger, color could be deeper according to weight of priority and time left to finish
3.1.2 biggest task can only occupy at most 1/5 of this top section
3.1.3 left side, in the middle, there is an arrow pointing to left, meaning to scroll to left to view more tasks in execution, when there are more tasks to display
3.2 next section below the top section is used to display the tasks supposed to start, but not yet. call it "start delayed".
3.3 next secion below "start delayed" is called "start next day" for tasks to start tomorrow
3.4 next secion below "start next day" is called "start in two days" for task to start the day after tommow
4. top right corner has menu icon to bring out more menu items
4.1 completed tasks
4.2 import from calendar
4.3 exprt to calendar

it shall has another screen to create task, in includes these fields:
- description, text box, but limited to 500 characters
- start time, include date and time
- end time, optional, estimated finish time
- priority: urgent 0, regular 1
- status: created, in execution, completed, abandoned
- creation time
- finish time

It has another screen to view tasks:
- display in calendar view
- on the calendar, by default display everything with status marked with different colors, such as green for in execution, grey for completed
- filter to choose status to display
